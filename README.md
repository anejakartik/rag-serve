# rag-serve — Production RAG microservice

**One-liner:** Production-ready RAG microservice with retrieval, caching, and LLM orchestration.

## Architecture
- Ingest -> Vectorize -> Vector DB -> Retrieval -> LLM -> API

## Stack
FastAPI, LangChain, Pinecone/Weaviate, Docker, Terraform, GitHub Actions
